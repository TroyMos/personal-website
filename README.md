# troymoslemi.com

Personal Angular site for Troy Moslemi.

## Run locally

Install dependencies with your package manager, then start Angular:

```sh
npm install
npm start
```

## Personal assets

- The site portrait is `public/assets/troy-moslemi-photo.jpeg`.
- Replace `public/assets/troy-moslemi-resume.pdf` with the final resume PDF. Keep the filename if you want the download link to keep working unchanged.
